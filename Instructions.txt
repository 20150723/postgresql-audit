1. Shut down the database first:
        /etc/init.d/postgresql-9.5 stop

2. Install the files into the database's directory:
        cp ./isecg_audit.so /usr/pgsql-9.5/lib/isecg_audit.so
        cp ./isecg_audit.control /usr/pgsql-9.5/share/extension/
        cp ./isecg_audit--1.0.sql /usr/pgsql-9.5/share/extension/

3. Edit the PostgreSQL configuration file, /var/lib/pgsql/9.5/data/postgresql.conf

        Find the line with 'shared_preload_libraries' and change it to read:

        shared_preload_libraries = 'isecg_audit'

4a. To enable logging to the Unix domain socket, add this line:

	isecgaudit.json_unix_socket = 1

4b. To enable logging to a file, add this line:

	isecgaudit.json_file = 1

6. Be sure the sensor is running.

7. Restart the database:

        /etc/init.d/postgresql-9.5 start
